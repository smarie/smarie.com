## Getting started memo

### Install yuicompressor

 * install nodejs for windows
 * add %appdata%\npm;%ProgramFiles%\nodejs; to path
 * `npm install -g yuicompressor`

### Install bundle and dependencies 
 
```bash
> gem install bundle
> bundle install
```
